# Republic AI Testnet v0.1.0

We're excited to announce the launch of Republic AI Testnet! 

## Quick Start

```bash
docker pull ghcr.io/republicai/republicd:0.1.0
```

## Network Information

| Property | Value |
|----------|-------|
| Chain ID | `raitestnet_77701-1` |
| EVM Chain ID | `77701` |
| Denom | `RAI` (18 decimals) |
| Min Gas Price | `250000000arai` |

| Service | URL |
|---------|-----|

| REST API | https://rest.republicai.io |

## Installation Options

1. **Binary** - Direct download with state sync
2. **Docker** - Container-based deployment
3. **Full Sync** - Sync from genesis block
